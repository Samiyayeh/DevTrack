# DevTrack

A full-stack project built with React (Vite) and Node.js.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clean install dependencies for both frontend and backend:

   ```bash
   cd client
   npm install
   cd ../backend
   npm install
   ```

2. Start the development servers:
   - For backend: `cd backend && npm run dev`
   - For frontend: `cd client && npm run dev`
